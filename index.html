    Mini Traffic Light and Drawbridge Simulation

    Arduino IDE • Arduino Programming Language

    Under IT Elective 3: Internet of Things, the main objective of this subject is to utilize sensors to achieve a fully automated system that can be applied in the real world projects. Mini Traffic Light and Drawbridge Simulation is develop using Arduino IDE and Arduino Programming Language that is based on C++. With the integration of various sensors, such as Light Dependent Resistors and Ultrasonic Sensors, their main functions are to trigger light switches and operate motor for smart movable bridge. This is only a small prototype but when scaled into real world machineries, this can help day to day mechanical jobs with less monitoring and minimal human effort.

    The Chase - 2D Endless Running Game

    Unity 2D • C# Programming Language

    The Chase is a classic 2D endless running game and only have a single level since it was just for project purposes. It was developed using Unity and mainly uses C# as programming languge. It uses some Scripting API such as PlayerPrefs classes that can store variables according to Player preferences between game sessions. Scoring systems are realized by utilizing game properties such as colliders, contactCounts, rigidBody, etc., and uses basic Quaternion for navigation.
